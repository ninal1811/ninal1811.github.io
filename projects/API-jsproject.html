JavaScript - API Name Popularity Project

Throughout the course the year, I've learned about the fundamental concepts of programming through JavaScript. In this project I was working with an API dataset. The API dataset I've chosen was on Popular Baby Names. As of right now, the API dataset only included female names.

Example of Project:

Challenges:

One of the biggest challenges I encountered was the extension part. For the extension, I wanted the program generate a random baby name for those who need a baby girl name recommendation. When I tried to make the program to generate a random name, it was only able to output one name. Thus, I added a prompt for the user to input a number which then allowed the program to give a random name recommendation. At the beginning of this project, I was also unable to print the data in the API. This is because the data in the API is formatted as different objects that don't have a specific name that I could use to access the names in each object.

Next Steps:

My next step would be styling the program so that it would fit with the information that it is providing. As learn more about JavaScript, I would like to advance my program to make it more efficient yet beneficial where the styling is just right.
