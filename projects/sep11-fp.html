Web Application - Freedom Project

Throughout the course the year, I've learned about the fundamental concepts of programming through JavaScript. In this project, I am working with a Firebase Database known as Cloud Firestore. Firebase allows me to store the user's task(s) which is beneficial for my To-Do List.

Example of Project:


Challenges:

One of the biggest challenges I encountered was incorporating Firebase into my project. Since Firebase is considered one of the difficult tools to learn, it challeneged me as I had no idea what it was. After researching about Firebase, I noticed there are two types of Firebase Database that I would use, Realtime Database and Cloud Firestore. When I first incorporated Firebase into my project, nothing was being saved on the Cloud Firestore. So I tinkered more with my Firebase, which then I realized that the position of where the code is located has a huge impact in which it can affect whether or not the information is being saved.

Next Steps:

My next step would be advancing my Firebase in which each user would have their To-Do List rather than one public one. As learn more about JavaScript and Firebase, I would like to advance my program to make it more efficient yet beneficial.
